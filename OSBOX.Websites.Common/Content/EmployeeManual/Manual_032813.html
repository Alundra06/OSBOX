Payroll- SUTA(State Unemployment Tax Act)

State, Quarterly Basis, due date is end of month.

Objective group: Employers

Report department: Maryland State

Filling Frequency: Quarterly


Filling Due Dates: End of month

Steps:

  1. Go to Quickbook->Reports->memorized->Gross, write down the number "SUTA"
  2. Document in Word form("SUTA, F:/drive"), print and mail out to clients.
  3. Pay online, using B file. (F:/14/B-file logon list-2014).
  4. Go to Quickbook->home->Process payroll form->State form->SUTA-> file the reports, using file cabine
